## Idea

One day while coding the in Zed IDE, I came to learn about the option to track the Agent movement in the editor while it is editing, reading and navigating the code all by itself.
Then it hit me with the feeling what if my activity(as in clicks across all the website) can be save without much cognitive load to mark it and save it just like how bookmark thing works.
This idea was spawned out of laziness to bookmark things while you are actively researching/reading anything in the internet/browser. That is it!
